# ds-ts-algo

ds-ts-algo is a typescript written package that allow users to use different pre-made data structures and algorithms in their javascript/typescript projects. 

<br />

## Data Structure included:
  - Stack
  - Set
  - Queue
  - Linked List
  - Doubly Linked List
  - Graph

## Algorithms included:
  - ### Search:
    - Binary Search
    - Linear Search

  - ### Sort:
    - QuickSort
    - HeapSort
    - InsertionSort
    - SelectionSort
    - BubbleSort
